Setup

Does Cura Home needs network connection to work?

Cura Home system will transfer video, audio and the status of smart sensors to your mobile phone through WiFi network. Cura Home system supports 2.4GHz WiFi network.

Does Cura Home system needs to be tethered to the computer during the setup process?

Cura Home system can be easily setup by mobile app. There is no need to use computer or other complicate way to setup the Cura Home system.

What happens if my personal WiFi connection goes down?

Cura Home system must connect to WiFi network, so please recover your WiFi network as soon as possible and contact your internet service provider if necessary.
